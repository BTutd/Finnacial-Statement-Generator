Financial Statement Generator & Forecasting

A modern web application built with Next.js (App Router) for generating financial statements, analyzing transactions, and forecasting future financial trends. Designed to help businesses and individuals visualize, understand, and predict their financial performance.

Features

User Authentication: Secure login and registration with JWT.

Transaction Management: Add, edit, delete, and view income and expense transactions.

Financial Statements:

Income Statement

Balance Sheet

Cash Flow Statement

Summary view of finances

Forecasting: Predict future income, expenses, and profit trends using moving averages and regression models.

File Upload: Import transactions via CSV or Excel files.

Data Visualization: Interactive charts for income, expenses, and forecasts.

Export: Export statements as PDF or Excel (planned for future release).

Responsive UI: Mobile and desktop-friendly dashboard.

 Tech Stack

Frontend: Next.js 14+ (App Router), TypeScript, Tailwind CSS, Recharts

Backend: Next.js API Routes, Node.js

Database: Supabase

Authentication: Supabase(Oauth)

Utilities: CSV & Excel parsing, date & currency formatting
